# Maison Reserve 静奢限量商品预约站

## 主题定位

用于静奢限量商品预约、编辑电商专题、高定工作室新品释出、会员制私享看货与高客单价商品咨询页面。核心气质是克制、留白、真实库存状态、预约动作清晰，而不是炫技式奢华。

这个主题应生成完整纯前端单页站点，不是孤立组件展示。页面必须围绕一个明确业务对象组织叙事，例如限量手袋、珠宝、腕表、香氛、家具或高定服饰的预约咨询链路。

## 必须遵守

- 这不是组件库：不要把按钮、卡片、表格、导航拆散后机械拼装；必须先理解业务路径，再组织页面叙事。
- 必须结合业务：页面至少呈现一个限量商品、一个预约/看货动作、一个真实状态或指标，例如剩余件数、候补人数、可预约时段、手工周期。
- 禁止直接复制：不要照搬示例站的文案、版式和数据；只能借鉴审美原则、层级节奏、色彩关系与交互克制度。
- 禁止堆叠组件：不要为了显得丰富而连续堆卡片、徽章、分栏、图表；每个区块都必须回答用户决策问题。
- 禁止 Lorem ipsum、空洞口号、随机奢华词汇、廉价金色大渐变、厚重发光、玻璃拟态滥用、过度圆角和低对比文字。
- 禁止远程资源、外链脚本和依赖外部字体；使用完整 HTML、内联 CSS 与系统字体完成交付。

## 视觉原则

- 背景使用暖象牙、米杏、深棕、古金；金色只做强调线、主按钮、状态标记。
- 标题可用系统衬线字体，正文使用系统 sans-serif；保持杂志编辑感和清晰可读性。
- 大留白服务于商品价值和决策节奏，不是空页面。
- 商品图可用 CSS 构造抽象产品轮廓，但必须像服务真实商品，而不是随机装饰块。
- 动效要低幅度、低频率，并支持 `prefers-reduced-motion` 降级。
- 移动端优先保证预约按钮、库存状态和商品信息不溢出、不遮挡。

## 业务结构建议

1. 首屏：品牌、限量商品、剩余件数、核心预约动作。
2. 限量说明：编号规则、本周可预约时段、交付周期。
3. 工艺记录：材质、产地、顾问服务和真实限制。
4. 编号库存：以克制表格或状态列表说明可预约、看货中、候补、已锁定。
5. 预约表单：只收集必要字段，如称呼、偏好时段、意向编号、关注事项。

## 适合场景

- 限量包袋、珠宝、腕表、香氛、皮具、家具、高定服饰的预约/咨询页。
- 品牌编辑电商专题、季节胶囊系列、私享预览邀请页。
- 需要表达稀缺性、工艺、等待周期、会员预约的高客单价转化页面。

## 不适合场景

- 低价促销、秒杀、强折扣、大库存 SKU 列表页。
- 高频运营后台、复杂数据分析台、强工具属性 SaaS 页面。
- 年轻潮流、游戏、电音、赛博、强娱乐化视觉项目。
- 需要大量广告位、弹窗、强刺激转化的增长页面。

## 生成建议

先写清业务对象与转化目标，再设计页面：首屏说明商品与预约价值，中段呈现工艺/库存/时段，底部给出低压但明确的预约表单。所有装饰都必须辅助信息层级。最终产物应像一个可以交给品牌方预览的单页站，而不是设计系统样张。
