# 玄门秘境档案主题使用指令

你正在使用 `xianxia / Celestial Sect Archive` 主题。它不是组件库，不提供可直接堆叠的通用 UI 片段；它是一套面向真实业务网站的视觉与信息结构参考。

## 主题定位

为东方玄幻宗门任务、秘境探索档案、弟子修行进度、法器封存记录、灵脉风险巡检等产品场景生成完整网站。整体气质应克制、秩序化、有卷宗感：深墨青底、玉金线框、低饱和灵光、细密档案纹理、清晰状态语义。

## 必须做到

- 必须结合业务：例如宗门任务调度、秘境开放窗口、队伍资质、灵脉稳定度、封印风险、卷宗审核流。
- 必须生成完整页面，而不是孤立组件或展示稿。
- 必须让每个视觉区域承担业务职责：导航、任务筛选、状态总览、档案列表、风险提醒、行动入口都要可解释。
- 必须使用真实可读文案、指标、状态、表格行和输入内容，禁止 Lorem ipsum。
- 必须保持响应式：桌面端有明确左右信息架构，移动端单列不溢出。
- 必须具备 hover、focus、active 等基础交互状态，并支持 `prefers-reduced-motion` 降级。

## 禁止事项

- 禁止直接复制 `preview.html` 的结构、文案或样式；应提取审美 DNA 后按新业务重组。
- 禁止堆叠组件：不要把 hero、card、table、form 简单拼接成页面。
- 禁止把主题理解为组件库；不要输出“按钮集合”“卡片集合”“表格组件合集”。
- 禁止廉价游戏皮肤化：大面积高亮、厚重发光、随机符咒装饰、过度圆角、低对比文字都不符合主题。
- 禁止只有“高级感”没有业务信息层级。

## 审美 DNA

- 主体表面：深墨青 `rgba(16,29,25,0.88)`，带轻微透明与层叠边界。
- 主强调：玉金 `#dabf7c`，用于主行动、关键边框、当前状态。
- 辅助强调：青玉 `#8fc7aa`，用于稳定、可进入、低风险信息。
- 字体节奏：标题可用宋体/衬线气质，正文保持系统 sans-serif 易读。
- 构图语法：卷宗、任务榜、秘境档案、灵脉观测、队伍编组共同构成完整业务闭环。

## 输出要求

输出单个完整 HTML 文件，内联 CSS，无远程资源，无构建依赖。页面必须像一个可以落地的东方玄幻业务系统，而不是风格演示图。
